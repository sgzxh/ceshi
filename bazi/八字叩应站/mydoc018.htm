第17篇 辛骑羊兔，富比陶朱

 

　　女命：民国六十八年（一九七九）六月卅日午时生

　　八字：己未年、辛未月、辛卯日、甲午时

　　大运：6壬申、16癸酉、26甲戌、36乙亥、46丙子、56丁丑

　　逢甲己两年立冬后十二天交脱大运

　　此命造是住在嘉义大林陈太太的外孙女，陈太太参加叩应节目想要了解外孙女在丁丑年大学联考的考运如何？运途如何？

　　“陈太太妳的外孙女八字本命星属辛金，生于六月底，命局不强不弱，财杀两停，格局不错。”唐老师首先论命格高低。

　　辛金生于未月，己土秉令，年柱再逢己未，己土通透有力，土旺金相。但辛金逢土厚埋金，污金不耀，不喜再有旺土生扶；虽然日元坐财，时柱天干甲木财星可疏土，但甲木究竟有合己土之嫌，疏土无力。且木从火势，助长火势，全局不见滴水，旺火反而足以销熔珠玉。因此，急须壬水冲刷其土、制其火，土润则木生，亦可淘金显珠玉之光，两全其美。若以强弱均衡、财杀两停推论为佳造，实有偏颇。命局应以“调候为急”，失去调候用神或用神无气乃清枯之命，穷通与否则系乎于运。

　　“做事情懒散，腰骨软（慵慵懒懒），不积极，讲话慢吞吞。”

　　这是印星强旺缺乏食伤吐秀，个性行为的具体表现。

　　“论考运，癸酉运不错，因为有食神智慧星制伏杀星，而且酉金是日元辛金的禄地。但此运有不少黑狗兄（异性朋友）紧缠着，锲而不舍，恐影响学业。那是因为有三个桃花在作祟。”唐老师继续论着。

　　八字论断考运、学历应着重印星与食伤星，印星代表求知慾，食伤星则代表智慧才华、反应的灵敏度，当然不能忽略岁运所引起的变化。壬申运，伤官星自坐长生，喜用并临，辛金获壬水淘喜，钏钗脱颍而出，珠玉之光悉呈现，喜神照命，学业定然出众。

　　癸酉运，癸水食神星仍是文昌星，但癸水根基不足，比起壬申运壬水自坐长生，其力差矣。除非流年太岁使癸水通根有力，否则成绩表现远不及壬申运，何况酉金冲动夫宫卯木，并且引动午火杀星桃花。丙子年双辛争合一丙，子午卯酉四正桃花星全部引动，渊海子平女命总断歌云：“子午逢卯酉，定是随人走。”想跳脱三角恋爱的漩涡，诚属不易。丁丑年与月柱辛未反吟，杀星高透，食伤星癸水反而有被熬乾之危，若要抡元及第，科甲联登，理想太高恐将失望。

　　“论财利，须卅五岁以后才会有发展，卅五岁以前因行运于戍土火库之地，火土太旺则不生财。”

　　书云：“辛骑羊兔，透乙木者，富比陶朱。”辛未日生者，自坐财库；辛卯日生者，自坐财星。若有财星透干，命局调配得宜，其财富可以比拟陶朱公范蠡。命造本身是辛卯日生，时柱天干甲木财星透出，理应财富丰厚，殊不知，木能生火，火多木焚。甲戍运，午戍半合火，卯戍合火、未戍相刑，使原本已是火炎土燥的命局更形焦坼。“南木飞灰而脱体”甲木随时有气散南离之危。所以，甲戍运财利恐将如唐老师所言难有所得。

　　乙亥运，“木从水养”，财星如久旱逢甘霖，时雨滋苗，财业多进、谋事称心。

　　“论婚姻，偏财坐夫宫且带流霞，所嫁的丈夫较外向，家里待不住，而且丈夫容易带有赌博的坏习惯。廿五岁以后所嫁的丈夫会较勤劳。不过，四十五岁以后丈夫容易有外遇，必须与人共夫，因为夫宫坐桃花的关系。”唐老师明察秋毫，对婚姻作出非常贴切的判断。

　　推论丈夫外向，嗜赌成性，并非是夫宫带流霞的关系，笔者认为是夫宫卯木偏财及命局火土旺盛的关系。因为木的特性喜向阳伸展，可推论夫宫本质是喜拓展人际关系。偏财星乃慾望之神，所追求的是四方财，再以夫星午火而论，局中土旺容易造成丈夫利慾薰心不知足，因此容易嗜赌成性。

　　四十六岁起行丙子大运，消息赋云：“丙子辛卯相逢荒淫滚滚。”辛卯日生逢丙子大运或流年即构成荒淫的条件，但问题是命造本身或是丈夫走桃花，则应以命局和大运流年的干支综合判断。丙子运，命运岁天干形成双辛争合一丙，丙火为正官夫星，是姊妹争夫之象；在地支，命局夫宫卯木与年月柱地支未土争合，午火杀星亦与两个未土争合，夫宫与夫星皆有妒合的现象，女命合多不为贵；子运，子水乃夫宫卯木的桃花星，子午冲，子水桃花星引动午火杀星，子卯刑，刑及夫宫。这些错综复杂的关系，在在显示夫宫、夫星皆呈现不稳定的状态，且离不开桃花星，因此推断丈夫容易有外遇是极为适当的。

　　任何一套论命术是有它的极限的，八字也不例外无论妻财子禄寿，乃至芝麻小事，想全部从八字命盘推论出来，是有它的困难。但我们可借助于“占卜”，笔者曾经在一次堪察阳宅中，屋主的女儿两天前才离婚，隔天刚离婚的丈夫后悔莫及，跪哭着求婚，屋主的女儿想论八字，看夫妻是否能再续前缘。像这样的问题，用占卜推测反而比八字论命来得快速而且明确，其他方面的问题，当然也可以借由占卜的卦象预测未来，如果您学会了占卜，在八字论命时可说多一层保护膜，使您论命时不致有所失，因为卦理与子平八字可说学理相通，而且可相辅相成。所以，家师，宋英成老师常将它比喻为陆、海、空三栖全方位整体作战。

　　陈太太叩应唐老师的命理节目，请教外孙女的考运如何，节目中唐老师大多谈到感情、婚恋的问题，主题考试能否上榜只是一笔带过，站在为人排忧解惑的命理师，此时要是能够借助易经占卜那岂非更完善。笔者自维譾陋愿藉此机会提出两个卦例，同样是考运、感情的问题愿供各为参考：

　　例一、事由：黄小姐想参加普考，胜算如何？

　　午 月 空：申酉

　　辛巳日 得：山雷颐之损卦

　　螣 兄 / 寅

　　勾 父 // 子

　　朱 财 // 戌〔世爻〕

　　青 财 // 辰

　　玄 兄 x 寅化卯

　　白 父 / 子〔应爻〕

　　占断说明：

　　占问考试以父母、官鬼为用神，公务人员尤应着重官鬼爻。黄金策云：“父旺官兴，姓标虎榜。”占问考试如果父母爻、官鬼爻旺相，生扶世爻或临于世爻、合世爻，则可联登甲第，金榜题名。

　　此卦卦中官鬼爻申酉金逢旬空，“空”即是“无”的意思，金虽长生于日辰巳火，沐浴于月建午火，但巳午火终究有克金之实。所以，申酉金实质上是“真空”。初爻和五爻父母爻子水逢月建冲克，称为“月破”。父母代表文书，可推论为黄小姐本身才识不足。卦中二爻兄弟发动且化进神，黄金策云：“兄弟同经，乃夺标之恶客。”代表竞争非常激烈，僧多粥少，代表功名的官鬼又逢旬空，我们综合研判，黄小姐难题雁塔，夺标无望。卦中父母爻双现，但逢月破，父母爻亦可类化为公司，黄小姐正因连续换两次工作，深觉私人公司不稳定，才想参加考试。

　　以易卦卦象而论，“颐”卦，乃动下颚吃食物之状。篆曰：“山下有雷，颐。君子以慎言语，节饮食。”引申为：占此卦要识时务，等待时机，谨言慎行。“损”卦，则是先舍后得，先苦后甘，虽然目前劳心劳力，但祇要充实自己，下苦心，终有喜悦的回报。

　　根据卦象及卦爻论断，笔者建议黄小姐，稍安勿燥，工作要有恒心，利用时间多充实自己，待来年再参加考试，以免徒劳无功。

　　例二、事由：李先生占与女友感情发展如何？

　　午 月 空：午未

　　辛卯日 得：泽山咸卦

　　螣 父 // 未〔应爻〕

　　勾 兄 / 酉

　　朱 子 / 亥

　　青 兄 / 申〔世爻〕

　　玄 官 // 午

　　白 父 // 辰

　　占断说明：

　　世爻为自己，应爻无女友。占恋爱以世应为主，视其对待关系再辅以官鬼、妻财爻做为判断。黄金策云：“应生世悦服成亲。”应爻未土生世爻申金，代表女友深爱着李先生。但未土旬空，恐怕女友虚情假意。五爻酉金逢日辰卯木冲克而暗动，酉金与世爻申金同为兄弟，应爻未土生世爻申金却被酉金阻挡于前，等于李先生的女友事实上较爱另一个男人。兄弟申金持世，李先生目前财务状况不佳。世爻申金与日辰卯木暗合，“如日辰合而世应比合，因人成事。”李先生内心正牵挂着“妻、财”的事情；卯申暗合也可推论为李先生与女友的交往是“暗通款曲”的不正常现象。黄金策云：“兄动而爻临玄武，须防劫骗之谋。”五爻酉金兄弟暗动逢勾陈，须防对方有骗取不动产之嫌。

　　李先生占的是“咸”卦，没有变卦，但五爻酉金逢日辰卯木冲克而暗动，暗动也等于发动之意。五爻动，则咸卦即变为雷山小过卦。易曰：“动为始，变为终。”咸卦象征少女与少男心心相印，心电感应充满罗曼蒂克的热恋，双方满足而得到喜悦。显示李先生与女友正沐浴于爱情的狂欢之中，而李先生却不知危机四伏。因为变卦及雷山小过卦，小过卦的卦形由外层四个阴爻包围两个阳爻，代表小人太多，凡事若超越分寸，猛然妄进将招致灾祸。

　　综合卦爻与卦象的推论，笔者告诉李先生，女友恐是为了骗取不动产虚情假意，您应专注于事业与家庭，否则秋天一到，容易中“仙人跳”而破财，女友身边的另一位男人，身材应比您瘦小。

　　事实上，李先生坦言，自己已有妻室、小孩，最近业务拓展不顺畅，发现女友另有新欢，却又一直吵着要买新宅，誓言与李先生相厮守，所以不知如何是好，才想占卦了解未来发展，想不到从卦象中能论断这么清处。
